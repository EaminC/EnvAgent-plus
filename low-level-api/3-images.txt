=========================================
IMAGES (LOW-LEVEL CLI API)
=========================================

For EnvAgent-plus, the image operations required are:
1. Listing available images
2. Viewing image details

These allow the agent to select a valid base image (e.g., CC-Ubuntu20.04)
when launching a bare-metal instance.

Advanced image management operations (uploading, sharing, editing, etc.)
are not required for the agentâ€™s functionality and are included only as an
optional appendix at the end.

Most image operations use the OpenStack Image service (Glance), and all
commands require sourcing the OpenStack RC file.


====================================================
1. LIST ALL IMAGES

    openstack image list

Input:
    None

    A table listing:
        - ID
        - Name
        - Disk Format

        - Status

2. VIEW IMAGE DETAILS
====================================================

Command:
    openstack image show <IMAGE>

Input:
    <IMAGE> may be:
        - image name (e.g. CC-Ubuntu20.04)
        - image UUID

Output:
    A detailed table of properties, including:
        - disk_format
        - min_ram, min_disk
        - size
        - visibility (public/private/shared)
        - owner
        - created_at / updated_at
        - checksum
        - container_format
        - metadata fields

====================================================
APPENDIX: ADVANCED IMAGE OPERATIONS (Optional)
====================================================

These operations are NOT required by EnvAgent-plus, but are included for
completeness and for users who may want to manage custom images.

A. Upload a new image:
    openstack image create --file <path> --disk-format <format> <name>

B. Download an image:
    openstack image save --file <local_file> <IMAGE>

C. Share an image publicly:
    openstack image set --public <IMAGE>

D. Share an image with a project:
    openstack image set --shared <IMAGE>
    openstack image add project <IMAGE> <PROJECT_ID>
    openstack image set --accept <IMAGE>

E. Modify an image:
    openstack image set <IMAGE> --name <new> --property <key=value>

F. Delete an image:
    openstack image delete <IMAGE_ID>
